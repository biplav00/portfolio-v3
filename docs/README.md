# Portfolio - Biplav Subedi

QA Engineer portfolio built with Astro + MDX.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Astro 5.x |
| Content | MDX + Content Collections |
| Runtime | Bun |
| Deployment | Vercel |

## Development

```bash
bun run dev      # Start dev server: http://localhost:4321
bun run build    # Build for production
bun run preview  # Preview production build
bun run lint     # Check lint
```

## Content

- **About**: `src/content/about/main.mdx`
- **Blog**: `src/content/posts/*.mdx`

## Project Structure

```
src/
├── content/        # MDX content
├── components/     # UI components
├── layouts/        # Page layouts
├── pages/          # Astro pages
└── styles/         # Global CSS
```
